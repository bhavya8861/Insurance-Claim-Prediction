# Insurance Claim Prediction 

##  Project Overview

This project aims to predict whether an insurance policyholder is likely to file a claim using Machine Learning techniques. The project involves data preprocessing, exploratory data analysis (EDA), feature engineering, handling class imbalance using SMOTE, model training, hyperparameter tuning, and performance evaluation to build an accurate insurance claim prediction model.

## Problem Statement

The objective of this project is to develop a classification model that predicts whether an insurance claim will occur based on customer demographics, policy information, and historical insurance data.

## Dataset

The dataset contains insurance-related information such as:

- Customer Demographics
- Policy Details
- Vehicle Information
- Premium Details
- Claim History
- Risk Factors
- Target Variable (Claim Status)
